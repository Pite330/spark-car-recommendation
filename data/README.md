# 数据说明

## 来源

当前推荐主表使用 [16888 车主之家](https://www.16888.com/) 的完整离线抓取，抓取日期为 `2026-07-11`。`scripts/fetch_16888_full_options.py` 保存车系、配置款、完整参数和月销量原始表；Spark 再从这些表中按“车系 × 能源类型”构建推荐记录。每条记录均保留车系页、参数页和销量页 URL。

采集脚本默认串行请求、设置间隔并限制采集范围，不绕过登录、验证码或访问限制。它不是全站爬虫，也不保证覆盖全部在售车型。若站点结构或条款发生变化，应暂停更新并重新确认采集方式。

## raw

`raw/16888_full/tables/` 包含 1620 个车系、19482 个配置款、3643134 条配置参数和 29859 条月销量。原始文件不直接修改，也不填补来源缺失的参数。`raw/16888_cars_snapshot.csv` 只保留为早期 190 条热门样本，不再作为推荐系统默认输入。更新命令：

```bash
make data-full
```

## processed

运行 `make spark` 后生成：

- `cars.csv`：Web 和推荐算法直接加载的标准主表；
- `cars.parquet/`：Spark 标准列式输出；
- `stats.json`：价格区间、能源和车身类型统计；
- `metadata.json`：运行时间、Spark 版本、数据源、输入校验值和行数。

当前 Spark 输出为 1573 个有效车系、1744 条“车系 × 能源类型”推荐记录；另有 2 条无有效厂商名称的聚合记录被删除。价格直接采用来源页面展示的万元指导价；最近月销量用于计算对数归一化热度，同一车系的不同能源记录共享车系月销量。来源没有稳定续航字段，缺失项由推荐算法自动取消，禁止主观补齐。

该快照仅用于课程项目与技术演示，不代表实时、完整或可用于交易的汽车报价。仓库公开发布、扩大采集范围或商业使用前，应重新核查来源站点条款、版权和数据授权。
