# 数据说明

## 来源

当前原始快照来自 [16888 车主之家](https://www.16888.com/)，抓取日期为 `2026-07-10`。`scripts/fetch_16888_dataset.py` 从公开车系分类页选择有限数量的热门在售车系，再读取各车系公开参数接口和销量页面，提取车系、品牌、价格区间、级别、车身、能源、座位、年款、功率和最近月销量等已有字段。每条记录均保留车系页、参数页和销量页 URL。

采集脚本默认串行请求、设置间隔并限制采集范围，不绕过登录、验证码或访问限制。它不是全站爬虫，也不保证覆盖全部在售车型。若站点结构或条款发生变化，应暂停更新并重新确认采集方式。

## raw

`raw/16888_cars_snapshot.csv` 包含 144 个车系、190 条“车系 × 能源类型”记录。一个车系同时存在纯电、增程等动力版本时会拆成多条记录；原始文件不直接修改，也不填补来源缺失的参数。更新命令：

```bash
make data
```

## processed

运行 `make spark` 后生成：

- `cars.csv`：Web 和推荐算法直接加载的标准主表；
- `cars.parquet/`：Spark 标准列式输出；
- `stats.json`：价格区间、能源和车身类型统计；
- `metadata.json`：运行时间、Spark 版本、数据源、输入校验值和行数。

价格直接采用来源页面展示的万元指导价区间。最近月销量用于计算对数归一化热度；同一车系的不同能源记录会共享车系月销量。来源当前未稳定提供续航，绝大多数油耗字段也为空，推荐算法会自动取消对应评分项，禁止主观补齐。

该快照仅用于课程项目与技术演示，不代表实时、完整或可用于交易的汽车报价。仓库公开发布、扩大采集范围或商业使用前，应重新核查来源站点条款、版权和数据授权。
