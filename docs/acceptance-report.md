## 演示流程

1. 打开 `data/raw/china_car_prices_snapshot.csv` 和 `data/processed/metadata.json`，说明 Spark 输入与运行证据；
2. 执行 `make run`，打开 `http://127.0.0.1:5000`；
3. 输入 `15—20 万 / SUV / 纯电 / 城市通勤 / 5 座`；
4. 展示 5 款车型、分数、推荐依据和明确的放宽提示；
5. 勾选 2—3 款，展示参数表和雷达图；
6. 展示页面底部 Spark 统计图；
7. 保持大模型关闭再次推荐，证明主链路离线可用。

## 已知数据边界

- 来源提供的是美元起售价，当前按固定汇率换算，不是实时中国成交价；
- 来源没有销量、热度和油耗，相关字段保持为空，评分会自动重新归一化；
- 页面展示的车型参数以 2026-07-10 抓取快照为准，后续更新需重新运行抓取与 Spark 任务。
