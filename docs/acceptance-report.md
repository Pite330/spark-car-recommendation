## 演示流程

1. 打开 `data/raw/16888_full/manifest.json` 和 `data/processed/metadata.json`，说明完整原始数据与 Spark 运行证据；
2. 执行 `make run`，打开 `http://127.0.0.1:5000`；
3. 输入 `15—20 万 / SUV / 纯电 / 城市通勤 / 5 座`；
4. 展示 5 款车型、分数、推荐依据和明确的放宽提示；
5. 勾选 2—3 款，展示参数表和雷达图；
6. 展示页面底部 Spark 统计图；
7. 保持大模型关闭再次推荐，证明主链路离线可用。

## 已知数据边界

- 来源提供公开指导价区间和最近月销量，不是实时成交价或上牌量；
- 月销量按车系统计，同一车系的不同能源记录会共享该销量；
- 来源未稳定提供续航，缺失字段对应的评分项会自动取消并重新归一化；
- 页面展示的车型参数以 2026-07-11 抓取快照为准，后续更新需重新运行完整抓取与 Spark 任务。
